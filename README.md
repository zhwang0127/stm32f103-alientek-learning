# STM32F103 正点原子例程学习笔记

本仓库为个人跟随**正点原子**《手把手教你学STM32 HAL库》系列教程的学习记录，包含 27 个基于例程修改的 Keil 工程。

## 📺 教程来源

- **正点原子官方**：[openedv.com](https://www.openedv.com/)
- **视频教程**：《手把手教你学STM32 HAL库（全系列）》
- **开发板**：正点原子 STM32F103 战舰 / M48Z-M3 小系统板
- **原版例程**：正点原子随板配套资料

> 正点原子官方声明：**"配套程序视频文档资料全部开源免费"**
> 来源：[github.com/alientek-openedv](https://github.com/alientek-openedv)
