# 基于 STM32F103C8T6 的串口风扇调速器

## 项目技术报告

> **串口调速 · OLED 中文显示 · 互补 PWM 无级调速 · 启动补偿**  
> **2026 年 5 月 30 日**

| | |
|---|---|
| **开发平台** | 正点原子 M48Z-M3 最小系统板 |
| **主控芯片** | STM32F103C8T6 (72MHz · 64KB Flash · 20KB SRAM) |
| **硬件模块** | ATK-MB022 OLED · ATK-MB023 小风扇 · ATK-MB000 仿真器 |
| **开发环境** | Keil MDK5 + STM32 HAL 库 |

## 一、项目简介

本项目以 STM32F103C8T6 为主控，通过串口接收 PC 端命令控制有刷直流电机转速与方向，同时在 OLED 屏幕和串口调试窗口中实时显示中文状态信息。

| 外设 | 实现功能 |
|------|------|
| **USART1** (PA9/PA10) | 115200bps 串口通信，IDLE 中断不定长帧接收，printf 中文回显 |
| **TIM1** (PA8/PB13) | 互补 PWM 输出（20kHz），经 SS6285L H 桥驱动有刷电机正反转及无级调速 |
| **软件 I2C** (PA2/PA3) | 驱动 SSD1306 OLED 控制器，128×64 中文实时显示 |
| **PCtoLCD2002 字库** | 16×16 逐列式点阵，7 汉字：转速风扇正停反 |

## 二、硬件接口

### 2.1 模块引脚

ATK-MB022 和 ATK-MB023 均采用 BEE BLOCK 1×4 排针标准（2.54mm 间距）：

| 针脚 | OLED (MB022) | 风扇 (MB023) |
|:---:|:---:|:---:|
| ① | VCC (3.3V) | VCC (5V) |
| ② | GND | GND |
| ③ | SDA | FI（前进 PWM） |
| ④ | SCL | BI（后退 PWM） |

### 2.2 MCU 引脚分配

| MCU 引脚 | 连接 | 外设模式 | 说明 |
|:---:|------|------|------|
| PA2 | OLED SCL | GPIO 推挽输出 | 软件 I2C 时钟（PA2 无硬件 I2C） |
| PA3 | OLED SDA | GPIO 开漏输出 | 软件 I2C 数据（PA3 无硬件 I2C） |
| PA8 | 风扇 FI + LED0 | TIM1_CH1 (AF2) | 正转 PWM，与板载 LED 共用 |
| PB13 | 风扇 BI | TIM1_CH1N (AF2) | 反转 PWM |
| PA9 | 串口 TX | USART1_TX (AF3) | 发送至 PC |
| PA10 | 串口 RX | USART1_RX (AF3) | 接收自 PC（交叉连接） |

> 引脚经 STM32F103C8T6 数据手册 (DS5319) 与 M48Z-M3 IO 分配表三重交叉验证。

### 2.3 PWM 参数

| 参数 | 值 |
|------|-----|
| 定时器 | TIM1（高级，APB2 72MHz） |
| 预分频 | 71（72MHz ÷ 72 = 1MHz） |
| 自动重载 | 49（1MHz ÷ 50 = 20kHz） |
| 占空比 | 0~49（0%~98%，50 级） |

### 2.4 接线

```
OLED：VCC→3.3V  GND→GND  SDA→PA3  SCL→PA2
风扇：VCC→5V    GND→GND  FI→PA8   BI→PB13
串口：仿真器TX→PA10  仿真器RX→PA9  仿真器GND→GND（交叉！）
```

## 三、软件设计

### 3.1 串口命令协议

| 命令 | 功能 | 示例 |
|------|------|------|
| `0`~`50` | 设置 PWM 占空比 | `30` |
| `SPD:XX` | 同上，显式格式 | `SPD:25` |
| `STOP` | 停止电机 | `STOP` |
| `FWD` | 正转方向 | `FWD` |
| `REV` | 反转方向 | `REV` |

命令以 `\r\n` 结尾，USART1 通过 IDLE 中断检测帧结束。主循环轮询 `g_usart_rx_sta & 0x8000`，`strncmp` 匹配关键字，`atoi` 提取参数。

### 3.2 电机控制

```
正转：TIM1_CH1(PA8)=PWM，TIM1_CH1N(PB13)=0 → 电流 FI→电机→BI
反转：TIM1_CH1(PA8)=0，TIM1_CH1N(PB13)=PWM → 电流 BI→电机→FI
停止：两通道均关闭
```

20kHz 频率高于人耳可听范围，无电磁啸叫。

### 3.3 低速启动补偿

目标 PWM < 15 时，先输出 20（~40% 占空比）持续 80ms 冲破静摩擦，再降至目标值。效果：最低自启转速从 PWM 15 降至 PWM 3，且支持从停止直接跳变到任意转速。

### 3.4 OLED 中文显示

`atk_oled_show_chinese()` 从 `chinese_font.h` 字库中读取 PCtoLCD2002 生成的逐列式 16×16 点阵数据，逐点写入显存缓冲 `g_atk_oled_gram[128][8]`，通过软件 I2C 一次性刷新至 SSD1306。

**OLED 布局（128×64）**：

```
┌──────────────────────────┐
│ 风扇 UART Ctrl           │  中文"风扇"(16×16) + ASCII 标题
│ 转速: 60%                │  中文"转速" + 百分比数字
│ PWM:30     正转          │  PWM 值 + 中文方向(正转/反转/停)
└──────────────────────────┘
```

### 3.5 代码结构

| 文件 | 职责 |
|------|------|
| `main.c` (~170行) | 单 while(1) 直写：初始化→命令解析→OLED 刷新→串口回显 |
| `atk_oled.c` | OLED 驱动 + 逐列式中文渲染函数 |
| `chinese_font.h` | 7 字点阵字库（转速风扇正停反） |
| `myiic.c` | 软件 I2C 模拟（~250kHz） |
| `atk_fan.c` | 风扇控制 + 启动补偿 |
| `atim.c` | TIM1 互补 PWM 初始化 |
| `usart.c` | 串口初始化 + printf 重定向 + IDLE 中断接收 |

代码风格与 keil_project 下已有工程一致：简洁头部注释 + 纯 main() + while(1) 直写，无冗余函数和打印。

| PWM | 占空比 | 表现 |

## 五、总结

### 5.1 技术要点

- USART1 + IDLE 中断实现不定长串口命令帧接收
- TIM1 互补 PWM 经 SS6285L H 桥驱动有刷电机双向运行
- 软件 I2C 驱动 SSD1306 OLED（PA2/PA3 无硬件 I2C）
- 启动补偿脉冲克服静摩擦，最低自启转速降至 PWM 3
- PCtoLCD2002 逐列式 16×16 点阵字库实现 OLED 中文显示

### 5.2 改进方向

| 方向 | 说明 |
|------|------|
| 闭环控制 | 加装霍尔传感器/编码器实现 PID 转速闭环 |
| UI 增强 | 进度条、转速曲线 |
| 参数保存 | Flash 存储速度值，上电恢复 |
| 协议扩展 | MODBUS-RTU 或二进制协议 |
