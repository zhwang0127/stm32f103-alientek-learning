# 风扇串口调速系统 V1.1

## 硬件平台
- MCU: STM32F103C8T6 (M48Z-M3最小系统板)
- 显示: ATK-MB022 OLED模块 (0.96寸 128x64, I2C接口)
- 驱动: ATK-MB023 小风扇模块 (有刷电机, TIM1互补PWM)
- 调试: ATK-MB000 BEE BLOCK仿真器 (DAP-Link + CDC虚拟串口)

## 引脚连接
| 模块       | 引脚  | 功能        | 备注 |
|-----------|-------|-------------|------|
| OLED SCL  | PA2   | I2C时钟 (软件模拟) | |
| OLED SDA  | PA3   | I2C数据 (软件模拟) | |
| 风扇 FI   | PA8   | TIM1_CH1 正转PWM  | 与LED0共用, 转速越高LED越亮 |
| 风扇 BI   | PB13  | TIM1_CH1N 反转PWM | |
| 串口 TX   | PA9   | USART1_TX → PC RX | |
| 串口 RX   | PA10  | USART1_RX ← PC TX | 仿真器TX接PA10, RX接PA9(交叉!) |

## 串口设置 (115200 / 8N1 / 无校验 / 无流控)
## 串口命令 (以\r\n结尾)
| 命令       | 功能             | 示例       |
|-----------|-----------------|-----------|
| 0~50      | 设置PWM占空比     | `30`      |
| SPD:XX    | 同上             | `SPD:25`  |
| STOP      | 停止风扇         | `STOP`    |
| FWD       | 正转方向         | `FWD`     |
| REV       | 反转方向         | `REV`     |

## 实测调速范围
| PWM | 表现 |
|:---:|------|
| 0 | 停止 |
| 1~2 | 需手推 (静摩擦力过大, 开环控制极限) |
| 3~45 | 自由无级调速, 支持任意跳变 |
| 46~50 | 最大转速 |

> 启动补偿: 目标PWM<15时自动先输出~40%占空比持续80ms冲破静摩擦, 再降到目标值

## 编译下载
1. Keil MDK5打开 `Projects\MDK-ARM\atk_f103.uvprojx`
2. 编译 → 下载到STM32F103C8T6
3. 串口助手 115200/8N1, 发送命令

## OLED显示布局
```
┌──────────────────────────┐
│ 风扇 UART Ctrl     行1   │  中文"风扇" + ASCII标题
│ 转速: 60%          行2   │  中文"转速" + 百分比
│ PWM:30     正转    行3   │  PWM值 + 中文方向(正转/反转/停)
└──────────────────────────┘
```

## 注意事项
- 风扇模块VCC接5V (非3.3V)
- 串口必须交叉: 仿真器TX→PA10, 仿真器RX→PA9
- PA8同时驱动LED0和风扇FI: 风扇运行时LED亮度随转速变化(呼吸灯)
- PA2/PA3无硬件I2C, 使用软件模拟(STM32F103C8T6物理限制)
